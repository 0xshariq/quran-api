3. Get Ayah by Surah Number and Verse Number

Endpoint: GET /:ayah

Headers:

x-api-key : YOUR_API_KEY

Example Request:

GET /1:1

Sample Response:

{
  "number": 1,
  "text": "In the name of Allah, the Most Gracious, the Most Merciful."
}

4. Get Audio

Endpoint: GET /audio/:reciter/:surahNumber/:verseNumber

Headers:

x-api-key : YOUR_API_KEY

Example Request:

GET /audio/Alafasy_64kbps/1/1

Sample Response:

Audio file

Refer to /reciters for the reciter name. Make sure that the subfolder is passed in place of reciter in the route.

Endpoint: GET /para_page/paraNumber:pageNumber
