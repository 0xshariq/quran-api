📖 Quran API Documentation

Explore the Holy Quran through our RESTful API. Built with Node.js and Express.

Base URL

https://quran-api-ny11.onrender.com/api/v2

📚 Endpoints

1. Get All Surahs

Endpoint: GET /surahs

Example Request:

GET /surahs

Sample Response:

[
  {
    "number": 1,
    "name": "الفاتحة",
    "englishName": "Al-Fatiha",
    "englishNameTranslation": "The Opening",
    "numberOfAyahs": 7,
    "revelationType": "Meccan"
  },
  {
    "number": 2,
    "name": "البقرة",
    "englishName": "Al-Baqarah",
    "englishNameTranslation": "The Cow",
    "numberOfAyahs": 286,
    "revelationType": "Medinan"
  },
  ...
]

2. Get Surah by Number

Endpoint: GET /surah/:{surahNumber}

Example Request:

GET /surahs/1

Sample Response:

  "name": "الفاتحة",

3. Get Ayah by Surah and Ayah Number

Endpoint: GET /{surahNumber}:{verseNumber}

Example Request:

GET /1:1

Sample Response:

{
  "number": 1,
  "text": "بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ",
  "numberInSurah": 1,
  "juz": 1,
  "page": 1,
  "surah": {
    "number": 1,
    "name": "الفاتحة",
    "englishName": "Al-Fatiha"
  }
}
